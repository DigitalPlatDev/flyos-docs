# Code Server on FlyOS
FlyOS comes with Code Server built-in, which makes coding and text editing convenient. If you enjoy coding, you can access the FlyOS Code Server from any computer, enabling you to code anytime, anywhere, and securely store it on your portable device. (like your phone)    
**Default on port 5004, Default password: ```flyos```**
